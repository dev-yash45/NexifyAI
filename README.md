# 🚀 AI Career Suite

An AI-powered career development platform that helps users improve their resumes, optimize LinkedIn profiles, identify skill gaps, and receive personalized career growth recommendations.

## ✨ Features

### 📄 Resume Intelligence
- ATS Score Analysis
- Resume Level Detection
- Strength & Weakness Analysis
- Missing Skills Detection
- Recommended Skills
- Interview Readiness Score
- AI Verdict
- Personalized 30-Day Growth Roadmap
- Interactive Analytics Dashboard

### 🔗 LinkedIn Intelligence
- Profile Score Analysis
- Headline Optimization
- About Section Optimization
- Keyword Optimization
- Network Strength Analysis
- Growth Suggestions
- AI Verdict
- Interactive Performance Dashboard

### 🎯 Skill Gap Analyzer
- Career Matching
- Skill Gap Detection
- Missing Skills Recommendations
- Personalized Learning Path
- Career Readiness Assessment

### 📊 Analytics Dashboard
- Interactive Graphs
- Dynamic Career Insights
- Performance Tracking
- Growth Potential Analysis
